# Image Folder

Place your images here!

## Required Images:

### 1. Hero Image (Homepage)
**File**: `lip-filler-nyc.jpg`
- Size: 600-800px wide
- Aspect ratio: 4:5 (portrait)
- Shows lip filler or aesthetic treatment results

### 2. Gallery Images
Create a `gallery/` subfolder and add before/after images

### 3. Procedure Images
Create a `procedures/` subfolder for individual treatment photos

---

**Note**: Next.js automatically converts all images to WebP format for better performance!
Just use regular JPG or PNG files.
